== Installation ==

1. New BPS Pro installation:
2. IMPORTANT Uninstall BPS free first.
3. Use the WordPress Upload Zip installer to install the BPS Pro zip file.
4. Activate BPS Pro on the Activation page.
5. Run the Pre-Installation Wizard and Setup Wizard.

1. BPS Pro Upgrade installations/WordPress Dashboard installation: 
2. Upgrade BPS Pro the same as any of your other plugins.

1. BPS Pro Upgrade installations/BPS Pro built-in Upload Zip installer:
2. Download the bulletproof-security.zip file to your computer.
3. Go to the BPS Pro Upload Zip Install menu page and use the BPS Pro Upload Zip installer.
